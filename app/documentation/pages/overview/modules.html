Modules

What is it

  • A JavaScript file that you can use to expose properties or functions for later use in your application.
  • Modules can help making your code modular and reusable.
  • Wakanda modules follow the CommonJS standard. ( The same as NodeJS )

How to

The folder backend/modules inside your Wakanda project is where you usually create your modules. It contains a default module module-example.

As the example above shows, to import the content of an existing module defined inside the modules folder of your backend, we simply use the function require with a string identifying the path to the module we want to use.
