Wakanda User Space

The main view of the Wakanda User Space gives you access to the list of your own and shared cloud subscriptions and On-premise applications.


  • Cloud Applications : Each application is pointed out by its domain name followed by some details such as the region the application is created in as well as the offer type and the owner if the application is not yours. it also shows quick action buttons to preview the app, delete it or get its detailed view (A quick upgrade button is available for free applications).

  • On-premise Applications : Each application is pointed out by its name followed by some details such as the offer type and the owner of the subscription.
