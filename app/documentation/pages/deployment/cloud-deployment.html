Cloud Deployment

Creating your first application

Once you have created an account in the Wakanda User Space connect to the dashboard, click on the New Subscription button to open the application creation wizard.

Start by choosing a plan from the Production Environment suggestions. In our example, we will be creating a free application. To proceed, click on the Free application button.

By default, your application will be created in the US. If you prefer a different region, you can select it from the list.

As of the moment, only the US region is available.

Choose a valid application name:

Optionally, you can add custom domains to your application’s configuration.

You are also invited to provide your profile’s information

Before proceeding to the next step, all the required fields must be filled.

Finally you must agree to our Terms of Service before proceeding to the payment panel.

Even if you have chosen a free trial plan, you need to validate your Paypal account..

Publishing & updating your application

Using Wakanda Studio

In order to deploy your solution to your cloud application using Wakanda Studio follow these steps :

Create a repository

From the solution explorer, click on to open Git. Then, click on Create Git Repository.

Commit your changes

Add changes to your stage and commit:

Add a remote repository

Retrieve your remote repository URL from Wakanda Cloud

Setup

Click on to open the remote repositories panel.

And setup your remote settings:

IMPORTANT: the password will be saved as plain text in the .git/config file. You can however leave the Username and Password fields empty and cache your password in Git..

Mac OS X

git config --global credential.helper osxkeychain

Deploy

Click on Deploy from the Studio toolbar:

You can now push, for example, on your production branch:

Using Command line

Push your solution to the remote repository by using the following command

 $ git push https://[my-application-name].us-git.wak-apps.com master
If your local repository is using a branch name other than master, you must use the following command to push your solution to the remote repository: 
$ git push https://[my-application-name].us-git.wak-apps.com [local-branch-name]:master

Clone From your Cloud Application

Using Wakanda Studio

Open The Git Panel and select your Remote Git Repository, then click on Pull:

Using the command line

Clone your application from your Cloud remote repository by using the following command

$ git clone https://[my-application-name].us-git.wak-apps.com
